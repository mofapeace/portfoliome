# Godlove Mofa - Personal Portfolio

A modern, responsive, and cybersecurity-themed personal portfolio built with React and Vite. This portfolio showcases my skills as a frontend developer with a defender's mindset, bridging UI/UX and security tooling.

## 🚀 Features

*   **Terminal-Themed Hero Section:** A unique introduction mimicking a Linux terminal, complete with mock `nmap` scan outputs and a `mofa@host:~$ sudo run` aesthetic.
*   **Ubuntu-Style Sidebar Navigation:** A sleek, icon-based sidebar for easy navigation across the portfolio (Home, About, Projects, Services, Contact) with smooth scrolling.
*   **Interactive About Section:** Animated counters for years of experience, projects completed, and clients served, along with a breakdown of core skills (Frontend, Linux, Pentesting, Design, Network Design).
*   **Services Showcase:** An interactive, accordion-style section detailing services offered:
    *   Frontend Development
    *   OS Install & Dual Boot
    *   Graphics & Design
    *   Network Design
*   **Fully Functional Contact Form:** Integrated with [Web3Forms](https://web3forms.com/) for seamless email forwarding straight from the site.
*   **Responsive Design:** Fully responsive layout with a toggleable mobile sidebar for optimal viewing on any device.

## 🛠️ Technologies Used

*   **React 19:** Component-based UI development.
*   **Vite:** Lightning-fast frontend build tool.
*   **Tailwind CSS:** Utility-first CSS framework (configured alongside custom CSS).
*   **FontAwesome:** For sharp and scalable vector icons.
*   **Vercel Analytics:** For privacy-friendly site analytics.
*   **React Anchor Link Smooth Scroll:** For smooth navigation between page sections.

## 📁 Project Structure

```text
src/
├── assets/          # Images, logos, and static assets

│   ├── Footer/      # Site footer

│   └── Services/    # Interactive services accordion
├── App.jsx          # Main application component routing all sections
├── index.css        # Global CSS styles
└── main.jsx         # React application entry point
```

## 💻 Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository** (if applicable):
    ```bash
    git clone <your-repo-url>
    cd portfoliome
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
